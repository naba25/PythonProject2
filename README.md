# PythonProject2
PyChat is a Python-based offline intelligent assistant built with Tkinter and Python libraries. It works like a lightweight Jarvis-style chatbot that can answer questions, solve math problems, provide system commands, and search information using Wikipedia and web tools


Tech Stack :
Python 3
Tkinter (GUI)
pyttsx3 (Voice Engine)
Wikipedia API
Webbrowser module
JSON (Data Storage)
Regular Expressions
Threading
